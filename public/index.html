Description

Nest framework TypeScript starter repository.

Installation

Clone project

$ gh repo clone armancj/teslo-shop

1. Clone and change .env.template file to .env

2. Insert data to environment variables .env

3. Raise database

$ docker-compose up -d

4. Install NodeJS dependence

$ yarn install

Running the app

# development
$ yarn run start

# watch mode
$ yarn run start:dev

# production mode
$ yarn run start:prod

Run seed
